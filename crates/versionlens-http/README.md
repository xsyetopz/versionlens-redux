# versionlens-http

HTTP settings, request headers, agent configuration, and transport helpers used by registry providers.

## Role

HTTP configuration and client support for the Rust workspace in VersionLens Redux. This crate is part of the repository implementation and is not documented as an independent public API.

## Validation

```bash
cargo test -p versionlens-http
```

Run `bun run check` from the repository root before merging broad workspace changes.
